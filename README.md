# 日本の自治体による 3D 点群オープンデータ一覧

自治体等が公開している **オリジナル 3D 点群データ（建物・樹木などを含む点群）** を、都道府県別に整理したリポジトリです。

- DEM / DSM などの標高モデルは含めない
- LAS / LAZ / MMS / ドローン点群などオリジナル点群のみ対象

## データ構成
- data/pointclouds.md — Markdown 一覧
- data/pointclouds.csv — 機械可読データ
- docs/index.md — GitHub Pages（任意）
