# 日本の自治体による 3 次元点群オープンデータ一覧

自治体等が公開している **3 次元点群データ（オリジナルデータ: 建物・樹木などを含む点群）** を、都道府県別に整理したリポジトリです。

- DEM / DSM などの標高モデルは含めない
- LAS / LAZ / 航空レーザ測量/ MMS / ドローン点群などオリジナルデータのみ対象

## データ構成

- data/pointclouds.md — Markdown 一覧
- data/pointclouds.csv — 機械可読データ
- docs/index.md — GitHub Pages（任意）
